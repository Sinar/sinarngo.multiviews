sinarngo.multiviews Installation
--------------------------------

To install sinarngo.multiviews using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``sinarngo.multiviews`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        sinarngo.multiviews

* Re-run buildout, e.g. with:

    $ ./bin/buildout
